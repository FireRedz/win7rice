  152. -Changing Your Icons-
  153. The easiest way to change your program icons individually, without any tools or software, is to edit a shortcut. What I recommend doing is navigating to the programs root directory, and creating the shortcut there so there is no confusion. Creating a shortcut also gives you the added bonus of naming it whatever you want, and even setting launch options. Right click on the Icon>Properties>Change Icon. Use your desired .ico file. Pin this shortcut to your taskbar/start menu! For other icons (library icons, task tray icons) please see the windows software section for tools to help you change them. Its a lot faster and easier than going through registries and replacing files manually. Look for 7tsp or 7conifier for icon packs, and browse for icon packs in the Icon section!

  168. -Making sure your taskbar is aligned properly, and slim as possible-
  169. Ever install appows and notice the icons or the start orb seem to be a little misaligned? This is just a glitch with the taskbar that is easily solved by extending it. Unlock your task bar, and drag it to the top of your screen. You can do this by left clicking and holding the bar near the center, and moving it to the edges of the screen. Drag it to the top along the edges this way. Once its on top, hover over the middle of the bar near the bottom border until you see the resize option for height ?. Drag it down, extending it to double the normal height. Move it back down to the bottom of your screen, then size it back down using the same ? option. Lock your task bar. You should be at the correct size and alignment now.

  296. ***Font Rendering***
  297. Fonts appearing to have a greenish tint or blurry in Firefox/Bblean/Windows? This is due to an acceleration error, probably caused by subpixel rendering or monospace fonts not being supported in smaller sizes. Windows Cleartype also causes issues with EVERY browser, and overall is complete shit compared to Linux or Mac/OSX font rendering. Read about Cleartype for details on how it works. Consider switching fonts if none of the fixes below work for you, or try and get a version of your font that is NOT ".tff" or true type.
  298.  
  299. The best and easiest option in my opinion is to install Mactype or gdipp. These are services that replace Windows Cleartype font rendering with Mac/OSX or Linux font type rendering. Mactype is more customizable and has bolder bolds, while Gdipp is easier to use and works immediately upon install.
  300. Go here for details.
  301.  
  302. Recommended fixes for Firefox/Nightly:
  303. In Firefox, type "about:config" in the address bar and hit Enter. Continue after the warning.
  304. In the Filter box at the top, type "gfx.direct2d.disabled". This value will be set to false by default.
  305. Right click that line and select Toggle. This will change the value to true. Restart Firefox.
  306. You may also need to disable hardware acceleration, which is under the Advanced Options tab. Try the Anti-Aliasing Tuner too, if you feel the need for more tweaking.
  307.  
  308. -Fonts in Bblean: The only fixes I have found documented involve removing anti-alias, or keeping cleartype enabled. Personally I've seen no font rendering issues with bblean, however I've seen it on a few desktops. Usually haxrcorp.tff is the culprit. Use Haxrcorp s8 instead.
  309.

  383. 3DCC is a small tool which allows you complete control over your windows classic color schemes, much more so than whats available through the standard windows display preferences.
  384.  
  385. Installing 3DCC:
  386.  
  387.   1. Put bbColor3dc.dll in your blackbox\plugins directory.
  388.   2. In your bblean menu, go to Settings>Config>Plugins and select "Add Plugin".
  389.   3. Navigate to the blackbox\plugins directory, and select "bbColor3dc.dll".
  390.   4. Load the plugin from the Settings>Config>Plugins menu.
  391.  
  392. To use 3DCC Themes with bblean, you need to make a folder in your bblean/styles directory named "3dc". After creating a theme with the program, save the file in the newly created 3dc folder. Make sure the 3dc file has the same name as your style. For example, bb_crim.3dc to work alongside bb_crim.style. You can then apply the theme by loading your style via the bblean menu. 3dc is intended for use along Windows Classic visual styles. If you get confused, a readme can be found in your bblean/plugins/bbColor3dc folder.

  420. Q: My Volume keys on my keyboard stopped working when I installed bblean. Is there a fix?
  421. A: Yes. Some kind anons found a fix for it that works, however it involves some scripting with Auto Hot Key. If you are unfamiliar, see the window tiling/autohotkey section.
  422. First, download NirCMD here. You can find links at the bottom of the page.
  423. Extract the exes to your Windows directory (C:\Windows).
  424. Create or add the following lines to your Auto Hot Key Script:
  425.  
  426. Volume_Up::
  427. run nircmd.exe changesysvolume 2000
  428. return
  429. Volume_Down::
  430. run nircmd.exe changesysvolume -2000
  431. return
  432. Volume_Mute::
  433. run nircmd.exe mutesysvolume 2
  434. return
  435.  
  436. The same method with nircmd also applies to bbkeys. The code for bbkeys would look something like this:
  437. KeyToGrab(Num9),  WithModifier(Win), WithAction(ExecCommand), DoThis(C:\Blackbox\usr\bin\nircmd\nircmd.exe changesysvolume -2000)
  438. KeyToGrab(Num0),  WithModifier(Win), WithAction(ExecCommand), DoThis(C:\Blackbox\usr\bin\nircmd\nircmd.exe changesysvolume 2000)
  439.  
  440. Just edit the key you want to use for volume controls, save the .rc and restart blackbox. If you wanted to use the buttons or sliders on a media keyboard, use bbkeyhook to define them.
  441. Run the script. You an also compile the script with Auto Hot Key, and drop it in your windows Startup folder. This will allow Windows to run it automatically on boot. Copy the compiled .exe to C:\Users\"username"\AppData\Roaming\Microsoft\Window s\Start Menu\Programs\Startup.

  443. Q: How do I extend the width of my bbleanbar?
  444. A: Ctrl+Right click>config>Width percent.
  445. Alternatively, edit the bbleanbar.rc by selecting "Edit settings" and edit the bbLeanBar.widthPercent value.
  446.  
  447. Q: I want my bbleanbar to stretch across both monitors. How do I do this?
  448. A: There are two ways to do this. The easiest (and recommended to those with equal resolution monitors) is to edit your bbleanbar.rc settings and override the max width percent by changing the value of "bbLeanBar.widthPercent" from 100 to 200. If you have uneven monitors (i.e. 1440x900 and 1366x768) you would use a value closer to 170 or so. Alternatively you can load in a separate bbleanbar and drag it to your other monitor. You can simply clone your current settings. To do this, follow these steps:
  449. Go to your bblean installation directory. Open the plugins folder, and find the bbLeanBar folder.
  450. Copy + paste it in the plugins directory, and rename the folder to "bbleanbar2".
  451. Visit Settings>Configuration>Plugins, and edit plugins.rc
  452. Add the name of your new bar under the current bblean bar entry. It should look like this:
  453. # - Integrated toolbar with tasks and tray
  454. plugins\bbLeanBar\bbLeanBar.dll
  455. plugins\bbLeanBar2\bbLeanBar.dll
  456.  
  457. Save As plugins.rc
  458. Restart blackbox.
  459. The bar should appear automatically, if not you should now have the option to load it within the plugins menu.
  460. Repeat the steps provided for however many you need. Please note: ctrl+right click your current bbleanbar to drag it around if you aren't seeing the new bar load. Chances are it loaded on top of the current bar, since you cloned your settings.
