  132. -Changing Your Icons-
  133. The easiest way to change your program icons individually, without any tools or software, is to edit a shortcut. What I recommend doing is navigating to the programs root directory, and creating the shortcut there so there is no confusion. Creating a shortcut also gives you the added bonus of naming it whatever you want, and even setting launch options. Right click on the Icon>Properties>Change Icon. Use your desired .ico file. Pin this shortcut to your taskbar/start menu! For other icons (library icons, task tray icons) please see the windows utilities section for tools to help you change them. Its a lot faster and easier than going through registries and replacing files manually.

  149. -Making sure your taskbar is aligned properly, and slim as possible-
  150. Ever install appows and notice the icons or the start orb seem to be a little misaligned? This is just a glitch with the taskbar that is easily solved by extending it. Unlock your task bar, and drag it to the top of your screen. You can do this by left clicking and holding the bar near the center, and moving it to the edges of the screen. Drag it to the top along the edges this way. Once its on top, hover over the middle of the bar near the bottom border until you see the resize option for height ?. Drag it down, extending it to double the normal height. Move it back down to the bottom of your screen, then size it back down using the same ? option. Lock your task bar. You should be at the correct size and alignment now.

  236. ***Font Rendering***
  237. Fonts appearing to have a greenish tint or blurry in Firefox/Bblean/Windows? This is due to an acceleration error, probably caused by subpixel rendering or monospace fonts not being supported in smaller sizes. Windows Cleartype also causes issues with EVERY browser, and overall is complete shit compared to Linux or Mac/OSX font rendering. Read http://en.wikipedia.org/wiki/ClearType for details on how it works. Consider switching fonts if none of the fixes below work for you, or try and get a version of your font that is NOT ".tff" or true type if you are on Windows.
  238.  
  239. For more details on Cleartype in Firefox, read: https://bugzilla.mozilla.org/show_bug.cgi?id=661471
  240.  
  241. What is subpixel rendering, and how can it effect me?
  242. http://www.usabilitypost.com/2011/02/08/please-stop-disabling-subpixel-rendering/
  243.  
  244. Recommended Extension for adjusting blurry fonts in FF/Nightly:
  245. https://addons.mozilla.org/en-US/firefox/addon/anti-aliasing-tuner/?src=userprofile
  246.  
  247. Recommended fix for Firefox/Nightly:
  248. In Firefox, type "about:config" in the address bar and hit Enter.
  249. Continue after the warning.
  250. In the Filter box at the top, type "gfx.direct2d.disabled".
  251. This value will be set to false by default.
  252. Right click that line and select Toggle. This will change the value to true.
  253. Restart Firefox.
  254.  
  255. Fonts in Bblean:
  256. The only fixes I have found documented involve removing anti alias, or keeping cleartype enabled. Personally I've seen no font rendering issues with bblean, however its been reported.
  257.  
  258. Global font rendering fixes:
  259. -Edit your font manually to remove cleartype from it completely.
  260. Visit: http://www.electronicdissonance.com/2010/01/raster-fonts-in-visual-studio-2010.html for details.
  261. -Edit registries to disable all font smoothing entirely.
  262. Visit: http://answers.microsoft.com/en-us/windows/forum/windows_7-desktop/disable-all-font-smoothing-in-windows-7-ie/f180e803-3317-4433-8fd2-63aadaecc2d2 for details.
  263. -Use Windows Style Builder to disable ClearType for a font of your choice in your theme.
  264. See "Windows programs of interest" in the catalog, and look for "Ave Style Builder" for a guide and download link.
  265.  
  266. !!!The best option in my opinion is to install Mactype or gdipp.!!!
  267. This replaces windows cleartype font rendering with Mac/OSX and Linux font rendering.
  268. Visit: http://www.askvg.com/how-to-get-mac-os-or-linux-style-font-smoothing-in-windows/ for details.
  269. Mactype is more customizable and has bolder bolds.
  270. Gdipp is easier to use, and works immediately upon install.
  271. The difference is immediately noticeable.
  272. Download links are near the bottom of the page.

  311. -Using Windows Classic + 3DCC Themes for Bblean-
  312. What is 3DCC?
  313. 3DCC is a small tool which allows you complete control over your windows color schemes, much more so than whats available through the standard windows display preferences.
  314.  
  315. Installation:
  316.  
  317.   1. Put bbColor3dc.dll somewhere in the blackbox\plugins folder.
  318.   2. Load it from the plugin-config menu, or add it's relative path to "plugins.rc".
  319.  
  320. To use 3DCC Themes with bblean, you need to make a folder in your bblean/styles directory named "3dc". After creating a theme with the program, save the file in the newly created 3dc folder. Make sure the 3dc file has the same name as your style. For example, bb_crim.3dc to work alongside bb_crim.style. You can then apply the theme by loading your style via the bblean menu. 3dc is intended for use along Windows Classic visual styles. If you get confused, a readme can be found in your bblean/plugins/bbColor3dc folder
  321.  
  322. Download: http://dl.dropbox.com/u/1840925/3dcc4-en.zip

  354. Q: Certain task tray icons I normally see aren't appearing (battery/wifi).
  355. A: Install bblean as a shell.

  357. Q: How do I remove taskbar/tray/status icons?
  358. A: The easiest way is to shift+right click the icons on your bar. Otherwise, ctrl+right click your bblean bar. Edit settings, and find this line "bbleanbar.item: Tray". Put a "!" in front of it so they won't load.
  359.  
  360. Q: How to I hide these workspace names (alpha) and the taskbar arrows?
  361. A: Edit your bbleanbar.rc and put a "!" infront of these lines:
  362. bbleanbar.item: WorkspaceLabel
  363. bbleanbar.item: WorkspaceButtonL
  364. bbleanbar.item: WorkspaceButtonR
  365. bbleanbar.item: WindowButtonL
  366. bbleanbar.item: WindowButtonR
  367.  
  368. Q: How do I remove the side bars/slits/analog clock (etc)?
  369. A: Unload the plugins under your configuration. The names are self explanatory.
  370. Alternatively, you can set them to autohide and adjust the sizes.
  371. (settings>config>plugins>load/unload)  
  372. (Restart blackbox after if the settings do not immediately apply.)
  373.  
  374. Q: My Volume keys on my keyboard stopped working when I installed bblean. Is there a fix?
  375. A: Yes. Some kind anons found a good fix for it that works beautifully, but it involves some scripting with Auto Hot Key. If you are unfamiliar, see the window tiling/autohotkey section(12).
  376. First, download NirCMD here:
  377. http://www.nirsoft.net/utils/nircmd.html you can find links at the bottom of the page.
  378. Extract the exes to your Windows directory (C:\Windows).
  379. Create or add the following lines to your Auto Hot Key Script:
  380.  
  381. Volume_Up::
  382. run nircmd.exe changesysvolume 2000
  383. return
  384. Volume_Down::
  385. run nircmd.exe changesysvolume -2000
  386. return
  387. Volume_Mute::
  388. run nircmd.exe mutesysvolume 2
  389. return
  390.  
  391. The same method with nircmd also applies to bbkeys. The code for bbkeys would look something like this:
  392. KeyToGrab(Num9),  WithModifier(Win), WithAction(ExecCommand), DoThis(C:\Blackbox\usr\bin\nircmd\nircmd.exe changesysvolume -2000)
  393. KeyToGrab(Num0),  WithModifier(Win), WithAction(ExecCommand), DoThis(C:\Blackbox\usr\bin\nircmd\nircmd.exe changesysvolume 2000)
  394.  
  395. Just edit the key you want to use for volume controls, save the .rc and restart blackbox. If you wanted to use the buttons or sliders on a media keyboard, use bbkeyhook to define them.
  396. Thats it! Run the script and enjoy your volume keys again.

  398. Q: How do I extend the width of my bbleanbar?
  399. A: Ctrl+Right click>config>Width percent.
  400. Alternatively, edit the bbleanbar.rc by selecting "Edit settings" and edit the bbLeanBar.widthPercent value.
  401.  
  402. Q: I want my bbleanbar to stretch across both monitors. How do I do this?
  403. There are two ways to do this. The easiest (and recommended to those with equal resolution monitors) is to edit your bbleanbar.rc settings and override the max width percent by changing the value of "bbLeanBar.widthPercent" from 100 to 200. If you have uneven monitors (i.e. 1440x900 and 1366x768) you would use a value closer to 170 or so. Alternatively you can load in a separate bbleanbar and drag it to your other monitor. You can simply clone your current settings. To do this, follow these steps:
  404. Go to your bblean installation directory. Open the plugins folder, and find the bbLeanBar folder.
  405. Copy + paste it in the plugins directory, and rename the folder to "bbleanbar2".
  406. Visit Settings>Configuration>Plugins, and edit plugins.rc
  407. Add the name of your new bar under the current bblean bar entry. It should look like this:
  408. # - Integrated toolbar with tasks and tray
  409. plugins\bbLeanBar\bbLeanBar.dll
  410. plugins\bbLeanBar2\bbLeanBar.dll
  411.  
  412. Save As plugins.rc
  413. Restart blackbox.
  414. The bar should appear automatically, if not you should now have the option to load it within the plugins menu.
  415. Repeat the steps provided for however many you need.

  418. A: Right click menu>bbstylemaker
  419. Alternatively, you can "edit your current style" and plugin hex colors and variables there.
  420. Save your style in your styles directory, and apply it when finished under the styles menu.

  426. A: You can do so by putting your wallpapers directory and image name in your current styles .rc settings manually, or by using bbstylemaker>other>image file name.

  432. (Still not displaying? Edit your blackbox.rc in notepad++. Change the encoding to UTF-8)

  434. Q: Firefox/my application window is weird looking/overlapping and has a flickering bar. What do?
  435. A: Unload/disable/uncheck/toggle bbleanskin under your configuration.
  436. (settings>config>bbleanskin>toggle skin)
  437. (settings>config>plugins>load/unload>bbleanskin)
  438. If you still want the skin enabled, do the following:
  439. Put these in your Exclusions for firefox/nightly:
  440. "firefox.exe"
  441. "xul.dll:MozillaHiddenWindowClass"
  442. "xul.dll:MozillaWindowClass"
  443. For Google Chrome, put: "chrome.exe"
  444. To turn bbleanskin off permanently, you can manually comment it out in your exclusions (blackbox>bbleanskin>edit exclusions to open the .rc). Just put a "#" in front of "plugins\bbLeanSkin\bbLeanSkin.dll". You can always uncomment it later if needed.

  446. Q: My Rainmeter skin is overlapping with my bbleanbar, what do?
  447. A: Change the width% of your bblean bar to match your skin. If you are using bb_bar, load
  448. the background bar on "normal", and set the bb_bar to "topmost" via your rainmeter settings.

  456. Q: Google Chrome is lagging/crashing on certain webpages ever since I started using Bblean. Whats going on?

  459. Make sure you disable the correct version.
  460. For bug tracking and other reports on this bug, see:
  461. http://code.google.com/p/chromium/issues/detail?id=161307
  462. You can also try navigating to: chrome://flags/ and disabling "GPU compositing on all pages" and "Threaded compositing". This has been reported as a successful fix.

  572. Q: I downloaded a skin for my music, but the music is not displaying. What do?
  573. A: It's probably not set for your music player. You need to edit the skin. Manage>Edit your skin>find the line where it says "Player Name" or "player" in the code, and enter the name of your player.   582. For Weather.com : http://xml.weather.com/weather/local/#WeatherCode#?cc=*&unit=#Unit#&dayf=6

  810. ***Setting your own Window Variables***
  811. "WinMove,A,, W,X,Y,Z"
  812. Where
  813. W= Distance of the window from the left side of the screen
  814. X= Distance of the window from the top of the screen
  815. Y= Total width of the window
  816. Z= Total height of the window
  817. Example:
  818. NumpadHome::
  819.             WinMove,A,, 20,29,1024,512

  823. ***VirtuaWin and Multiple Desktops***
  824. http://virtuawin.sourceforge.net/
  825. VirtuaWin is an application for the Windows user community that creates multiple desktop functionality often seen on Unix systems. It is simple, reliable and highly configurable, supporting up to 20 virtual desktops. You can set up custom window rules, hotkeys, and is skinnable with Rainmeter.
